# DBeat

A lightweight application to help you view, search and organise your Rekordbox recordings.

## W.I.P

This application is currently experimental and unstable.

## Running the code

To run a dev instance of dbeat run the following in the `dbeat` folder:

```bash
npm install
npm run tauri dev
```

## Build the code

To build an executable run the following:

```bash
npm run tauri build
```

This build creates an executable `.exe` for Windows and `.dmg` for Mac.
